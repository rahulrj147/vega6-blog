# Vega6 — Full-Stack Blog Platform (Assignment)

**Vega6** is a modern full-stack blogging application built using **Next.js 16 (Turbopack)** and **Node.js/Express**. It features a simple and premium UI, JWT authentication, and a dynamic writer dashboard.

---

### 🚀 Key Features
-   **Secure Auth**: JWT-based login/signup with silent token refresh.
-   **Full CRUD**: Create, read, edit, and delete blogs with image uploads (Cloudinary).
-   **Dashboard**: Centralized hub for blog stats and management.
-   **Premium UI**: Responsive design with glassmorphism and smooth animations using **Tailwind CSS 4**.

---

### 🛠️ Tech Stack
-   **Frontend**: Next.js 16, React 19, Tailwind CSS 4, Lucide React, Axios.
-   **Backend**: Node.js, Express, MongoDB (Mongoose), Cloudinary, JWT.

---

### 🏁 Quick Start

#### 🌐 Backend Setup
```bash
cd vega-backend && npm install
# Setup .env (see .env.example)
npm run dev
```
*Runs on `http://localhost:5010`*

#### 🎨 Frontend Setup
```bash
cd vega-frontend && npm install
# Setup .env.local (see .env.example)
npm run dev
```
*Runs on `http://localhost:3000`*

---

### 📂 File Structure
-   📁 `vega-backend/`: API server and database models.
-   📁 `vega-frontend/`: Next.js frontend application.
-   🔑 `.env.example`: Template for environment variables.
